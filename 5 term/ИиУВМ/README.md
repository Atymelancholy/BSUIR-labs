## Общий вид лабораторных работ при сдаче:
Реализовать пользовательский интерфейс по варианту с использованием любых фреймворков, библиотек. 
Каждый студент на первом занятии выбирает себе персонажа мультфильма, который не может повторяться в рамках всего потока.
Из вашего приложения будут запускать все лабораторные работы по принципу выбора окошка с нужной лабораторной работой. Анимацию выбора вы настраиваете самостоятельно. 
Недостаточно сделать окно, по примеру ниже. Должна быть реализована анимация с выбранным персонажем мультфильма.

![alt text](<Снимок экрана 2025-11-03 112513.png>)

Каждая свинка является кнопкой, которая при нажатии открывает одну из лабораторных работ:
- Энергопитание 
- Конфигурационное пространство PCI (Windows XP)
- Устройства хранения данных (Windows XP)
- Веб-камера
- USB-устройства
- Работа с интерфейсом Bluetooth
