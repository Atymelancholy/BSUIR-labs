## Задания на лабораторные работы по КиДСВТ

- Лабораторная работа №1 - Моделирование комбинационных схем. Построение проверяющих тестов
- Лабораторная работа №2 - Исследование генераторов псевдослучайных последовательностей и сигнатурных анализаторов
- Лабораторная работа №3 - Разработка цифрового двойника оперативного ЗУ, разработка тестов  и моделирование неисправностей
- Лабораторная работа №4 - Изучение встроенных и внешних программ диагностирования персональных компьютеров
